**handleCreateInstance(w, r):**
1. Extract Clerk claims: `clerk.SessionClaimsFromContext(r.Context())`. Get orgID, userID.
2. Call `s.db.EnsureOrgAndUser(ctx, clerkOrgID, clerkUserID, email)` to auto-provision org+user.
3. Decode request body into CreateInstanceRequest, validate.
4. Build `provision.ProvisionRequest` from the API request + auth context.
5. Call `s.engine.Provision(ctx, req)`.
6. On success, return 201 with InstanceResponse (via `instanceToResponse`).
7. On error: if price exceeded, return 409 ProblemDetail. Otherwise 500 ProblemDetail (never leak provider details in error message -- log internally, return generic error to customer per API-09).

**handleListInstances(w, r):**
1. Extract orgID from claims.
2. Parse PageParams from query string.
3. Decode cursor if present.
4. Call `s.db.ListInstances(ctx, orgID, cursorTime, cursorID, limit)`.
5. Check if len(results) > limit to determine has_more. Trim to limit.
6. Map each instance to InstanceResponse.
7. Encode cursor from last item's (created_at, instance_id).
8. Return PageResult[InstanceResponse] as JSON.

**handleGetInstance(w, r):**
1. Extract instanceID from path: `r.PathValue("id")`.
2. Extract orgID from claims.
3. Call `s.db.GetInstanceForOrg(ctx, instanceID, orgID)`.
4. If not found, return 404 ProblemDetail.
5. Return InstanceResponse.

**handleDeleteInstance(w, r):**
1. Extract instanceID and orgID.
2. Call `s.db.GetInstanceForOrg(ctx, instanceID, orgID)` to verify ownership.
3. If not found, return 404.
4. If already terminated, return 200 with current state (idempotent per INST-06).
5. Call `s.engine.Terminate(ctx, instanceID)`.
6. Re-fetch instance from DB.
7. Return 200 with updated InstanceResponse.

1. Return middleware that:
   - Reads `Idempotency-Key` header. If empty, pass through to handler.
   - Extract orgID from Clerk claims.

   - Call `db.GetIdempotencyKey(ctx, orgID, key)`.

   - If key found AND response_code is nil (in-progress): return 409 ProblemDetail "Request with this idempotency key is already in progress".
   - If key not found: call `db.CreateIdempotencyKey(ctx, orgID, key, hash)`. If ErrIdempotencyKeyExists (race condition), retry GetIdempotencyKey.
   - Wrap the ResponseWriter to capture status code and body.
   - Call next.ServeHTTP with wrapped writer.
   - After handler completes, call `db.CompleteIdempotencyKey(ctx, orgID, key, statusCode, body)`.

```go
// Auth + rate limiting middleware chain
clerkAuth := ClerkAuthMiddleware(deps.Config.ClerkSecretKey)
requireOrg := RequireOrg
rateLimiter := NewOrgRateLimiter(rate.Every(time.Second), 20) // 20 req/s burst per org

// Middleware chain helper
authChain := func(h http.Handler) http.Handler {
    return clerkAuth(requireOrg(rateLimiter.Middleware(h)))
}

// Instance CRUD routes
idempotency := IdempotencyMiddleware(deps.DB)
s.mux.Handle("POST /api/v1/instances",
    authChain(idempotency(http.HandlerFunc(s.handleCreateInstance))))
s.mux.Handle("GET /api/v1/instances",
    authChain(http.HandlerFunc(s.handleListInstances)))
s.mux.Handle("GET /api/v1/instances/{id}",
    authChain(http.HandlerFunc(s.handleGetInstance)))
s.mux.Handle("DELETE /api/v1/instances/{id}",
    authChain(http.HandlerFunc(s.handleDeleteInstance)))

// SSE status streaming
s.mux.Handle("GET /api/v1/instances/{id}/events",
    authChain(http.HandlerFunc(s.handleInstanceSSE)))

// Internal callback (localhost + internal token, NOT Clerk auth)
s.mux.Handle("POST /internal/instances/{id}/ready",
    LocalhostOnly(InternalAuthMiddleware(deps.Config.InternalAPIToken,
```

**internal/api/sse.go** -- New file:

Implement `StatusBroker`:
```go
type StatusBroker struct {
    mu          sync.RWMutex
    subscribers map[string][]chan StatusEvent // instance_id -> channels
}

type StatusEvent struct {
    InstanceID     string `json:"instance_id"`
    Status         string `json:"status"`          // external state
    InternalStatus string `json:"internal_status"` // internal state
    Timestamp      string `json:"timestamp"`       // RFC 3339
}
```

Methods:
- `NewStatusBroker() *StatusBroker`
- `Subscribe(instanceID string) chan StatusEvent`: create buffered channel (size 10), add to subscribers map, return channel.
- `Unsubscribe(instanceID string, ch chan StatusEvent)`: remove channel from subscribers, close it.
- `Publish(instanceID string, event StatusEvent)`: send event to all subscribers for this instance. Use non-blocking send (`select { case ch <- event: default: }`) to avoid blocking if a subscriber is slow.

Implement `handleInstanceSSE(w, r)`:
1. Check `http.Flusher` support. If not supported, return 500 ProblemDetail.
2. Extract instanceID from `r.PathValue("id")`.
3. Extract orgID from Clerk claims and verify org ownership by calling `s.db.GetInstanceForOrg(ctx, instanceID, orgID)`. Return 404 if not found. This prevents SSE subscription to instances you don't own.
4. Set response headers:
   - Content-Type: text/event-stream
   - Cache-Control: no-cache
   - Connection: keep-alive
   - X-Accel-Buffering: no (for nginx proxies)
5. Subscribe to status broker.
6. Defer unsubscribe.
7. Send current state immediately as first event.
8. Start keepalive ticker (30 seconds).
9. Select loop:
   - `r.Context().Done()` -> return (client disconnected)
   - `event from channel` -> write `event: status\ndata: {json}\n\n`, flush. Check write error -- if error, return (client disconnected).
   - `keepalive tick` -> write `: keepalive\n\n`, flush.
10. Set maximum connection duration: 30 minutes. After that, close the connection (client can reconnect). Use a timer in the select loop.
