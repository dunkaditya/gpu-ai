       - Use `text/template` (not html/template) per project convention (03-03 decision).
       - Template variables: .InstanceID, .ProxyHost, .FRPServerPort, .FRPToken, .RemotePort, .SSHAuthorizedKeys, .Hostname, .CallbackURL, .InternalToken.
       - Include fallback: try GitHub CDN first, then GpuctlPublicURL if set.
