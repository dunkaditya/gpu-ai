```go
type SpendingLimit struct {
    SpendingLimitID      string
```

Methods:
`GetSpendingLimit(ctx, orgID string) (*SpendingLimit, error)` — SELECT all columns WHERE org_id = $1. Return `db.ErrNotFound` if none.

`UpsertSpendingLimit(ctx, orgID string, monthlyLimitCents int64) (*SpendingLimit, error)`:
```sql
INSERT INTO spending_limits (org_id, monthly_limit_cents, billing_cycle_start)
VALUES ($1, $2, date_trunc('month', NOW()))
ON CONFLICT (org_id) DO UPDATE
SET monthly_limit_cents = $2,
    updated_at = NOW()
RETURNING *
```

`DeleteSpendingLimit(ctx, orgID string) (bool, error)` — DELETE WHERE org_id = $1, returns false if not found.

`UpdateSpendingLimitFlags(ctx, orgID string, notify80, notify95 bool, limitReachedAt *time.Time) error`:
```sql
UPDATE spending_limits
SET notify_80_sent = $2,
    notify_95_sent = $3,
    limit_reached_at = $4,
    updated_at = NOW()
WHERE org_id = $1
```

`ResetMonthlySpend(ctx, orgID string) error` — resets for new billing cycle:
```sql
UPDATE spending_limits
SET current_month_spend_cents = 0,
    notify_80_sent = FALSE,
    notify_95_sent = FALSE,
    limit_reached_at = NULL,
    billing_cycle_start = date_trunc('month', NOW()),
    updated_at = NOW()
WHERE org_id = $1
```

```go
    Stripe  *BillingService
    Logger  *slog.Logger
}
```

Define `BillingTicker` struct holding the deps.

`NewBillingTicker(deps TickerDeps) *BillingTicker`

`Start(ctx context.Context)` — runs in a goroutine, fires every 60 seconds:
```go
func (t *BillingTicker) Start(ctx context.Context) {
    ticker := time.NewTicker(60 * time.Second)
    defer ticker.Stop()
    for {
        select {
        case <-ctx.Done():
            return
        case <-ticker.C:
            t.runTick(ctx)
        }
    }
}
```

`runTick(ctx context.Context)` — the main tick logic. CRITICAL ORDER: limits first, then Stripe.

```
func (t *BillingTicker) runTick(ctx context.Context) {
    // Step 1: Get all active billing sessions
    sessions, err := t.db.GetActiveBillingSessions(ctx)
    if err != nil { log and return }

    // Step 2: Group sessions by org
    byOrg := map[string][]db.BillingSession{...}

    // Step 3: For each org — LIMITS FIRST
    for orgID, orgSessions := range byOrg {
        t.enforceSpendingLimit(ctx, orgID, orgSessions)
    }

    // Step 4: THEN Stripe reporting
    t.reportToStripe(ctx, sessions)
}
```

`enforceSpendingLimit(ctx, orgID string, sessions []db.BillingSession)`:
1. Call `t.db.GetSpendingLimit(ctx, orgID)` — if `db.ErrNotFound`, skip (no limit set).
2. Check if billing_cycle_start is in the past month; if new month, call `ResetMonthlySpend`.
3. Call `t.db.GetOrgMonthSpendCents(ctx, orgID, limit.BillingCycleStart)` for live spend.
4. Compute percentage: `pct = liveSpendCents * 100 / limit.MonthlyLimitCents`
5. Thresholds (only fire once each, tracked by notify_*_sent flags):
   - `pct >= 80 && !limit.Notify80Sent` → log warning, call `UpdateSpendingLimitFlags(notify80=true, notify95=false, limitReachedAt=nil)` (webhook future: log "SPEND_WARNING_80" for now)
   - `pct >= 95 && !limit.Notify95Sent` → log warning, call `UpdateSpendingLimitFlags(notify80=true, notify95=true, limitReachedAt=nil)`
   - `pct >= 100 && limit.LimitReachedAt == nil` → stop all running instances for this org (call `t.engine.StopInstancesForOrg(ctx, orgID)`), set `LimitReachedAt = &now`, log "SPEND_LIMIT_REACHED"
   - `limit.LimitReachedAt != nil && time.Since(*limit.LimitReachedAt) > 72*time.Hour` → terminate all stopped instances for this org (call `t.engine.TerminateStoppedInstancesForOrg(ctx, orgID)`), log "SPEND_LIMIT_AUTO_TERMINATE"

`reportToStripe(ctx, sessions []db.BillingSession)`:
1. For each session, compute unreportedSeconds = `int64(time.Since(session.StartedAt).Seconds()) - session.StripeReportedSeconds`
2. Group by org: accumulate unreportedSeconds + get `stripe_customer_id` from org record (`t.db.GetOrgStripeCustomerID(ctx, orgID)` — returns empty string if not set)
3. Build `[]MeterEventBatch` — skip orgs with no stripe_customer_id or 0 unreported seconds
4. Call `t.stripe.ReportMeterEvents(ctx, batches)`
5. For each session with reported seconds > 0, call `t.db.UpdateStripeReportedSeconds(ctx, session.BillingSessionID, unreportedSeconds)`
6. Use time.Now().Truncate(60*time.Second) as TimestampBucket for idempotency key

```go
limit, err := e.db.GetSpendingLimit(ctx, orgID)
if err == db.ErrNotFound {
    return nil // no limit set, allow
}
if err != nil {
    return fmt.Errorf("check spending limit: %w", err)
}
if limit.LimitReachedAt != nil {
    return ErrSpendingLimitReached
}
// Also check live spend in case ticker hasn't run yet
spendCents, err := e.db.GetOrgMonthSpendCents(ctx, orgID, limit.BillingCycleStart)
if err != nil {
    return fmt.Errorf("check spending limit: %w", err)
}
if spendCents >= limit.MonthlyLimitCents {
    return ErrSpendingLimitReached
}
return nil
```
